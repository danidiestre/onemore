# Renombrar el proyecto en Vercel

El repo en GitHub y la carpeta local ya están renombrados a **onemore**. Para que el nombre coincida en Vercel:

1. Entra en el **dashboard de Vercel**: [vercel.com](https://vercel.com)
2. Abre el proyecto (sigue vinculado al repo **danidiestre/onemore**).
3. Ve a **Settings** → **General**.
4. En **Project Name** cambia `drink-counter` por **onemore** (o el nombre que quieras).
5. Guarda.

La conexión con GitHub no se pierde al renombrar. Si tienes el proyecto vinculado con `vercel link`, sigue funcionando desde la carpeta **onemore**.

## Reabrir el proyecto en Cursor

La carpeta del proyecto ahora es:

```
/Users/danidiestre/dev/onemore
```

Abre esta ruta en Cursor (File → Open Folder → `onemore`) para seguir trabajando.
